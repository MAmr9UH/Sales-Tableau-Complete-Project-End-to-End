### Dashboard Development for the Project 

1. **Analyze the requirements**: Collect and understand the project requirements , Select the appropriate charts based on the analysis goals, Create mockups for the visualizations.
2. **Build a Data Source**: Connect with the data, create data model and rename fields and check the data type, understand the data.
3. **Build charts**: Use calculated fields or ensure all data needed for charts is present, Building the charts, Format: adding colors edit the axis and headers.
4. **Build a dashboard**: Create a mockup of the dashboard layout using Tableau's containers, create container structure, Add the charts created earlier to the dashboard: including filters, distributed objects, Adding Icons , Adding interactive filters .
